# webapp_skill

このリポジトリは、ワークショップで `skill` を使いながら Web アプリ作りを進めるためのサンプルです。

## まず知っておくこと

この workshop では、最初から全部を自分で書くよりも、`skill` にやりたいことを伝えながら進める使い方を想定しています。

`skill` は、作業の進め方を助けるためのガイドや補助機能です。  
まずは「何を作りたいか」「次に何をしたいか」を短く伝えるところから始めてください。

## skill の使い方

基本は、やりたいことを日本語でそのまま伝えれば大丈夫です。

例:

- 「プロフィール画面を追加したい」
- 「投稿一覧をカード風にしたい」
- 「このボタンをもっと目立つデザインにしたい」
- 「localStorage に保存できるようにしたい」
- 「今のコードを初心者向けに説明して」

必要なら、使いたい skill 名を一緒に書いても構いません。

例:

- 「`frontend-design` を使ってトップページを見やすくして」
- 「`nextjs-workshop-sns` を使って、SNS っぽい投稿画面を作りたい」

## workshop でのおすすめの使い方

1. まずは作りたい画面や機能を 1 つだけ決める
2. その内容を短い日本語で伝える
3. 返ってきた内容を見て、足したい点や直したい点を続けて伝える
4. 一度にたくさん頼まず、小さく区切って進める

## コツ

- 「何をしたいか」が入っていれば十分です
- 困ったら「初心者向けに説明して」と付けると進めやすくなります
- 見た目を整えたいときは、画面名と雰囲気を一緒に伝えると伝わりやすいです

例:

- 「新規投稿ページを、やさしい雰囲気で見やすくしたい」
- 「マイページを SNS っぽくしたい」

## 迷ったとき

何から始めればよいか分からないときは、まず次のように聞いてください。

- 「この workshop では次に何を作るのがおすすめ？」
- 「進めやすい順番で作業を分けて」
- 「今の状態から、次にやることを 3 つ出して」
